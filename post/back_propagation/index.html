Calculate Gradient for Back Propagation

PROJECTS

Back Propagation

July 5, 2020

As part of class, we looked at a simple Neural Net with one hidden layer. The Neural Net uses hyperbolic tangent for the input layer activation and sigmoid for the hidden layer activation. The forward pass is shown below:

$Z_{1} = W_1 X + B_1 \\\
A_1 = tanh(Z_1) \\\
Z_2 = W_2 A_1 + B_2 \\\
A_2 = (1 + e^{-Z_2})^{-1} \\\
J = -\frac{1}{n}\sum_{1}^{n}{(Y log(A_2) + (1-Y) log(1-A_2))} \\\ $

Our professor challenged us to derive the back propagation. I calculated the gradients of the cost function with respect to the weight and bias matrices, as shown below:

1.) $\frac{\partial J}{\partial Z_2} = \frac{\partial J}{\partial A_2} \frac{\partial A_2}{\partial Z_2}$

2.) $\frac{\partial J}{\partial A_2} = -\frac{Y}{A_2} + \frac{(1-Y)}{1-A_2} = -\frac{Y(1-A_2)}{A_2(1-A_2)} + \frac{A_2(1-Y)}{A_2(1-A_2)} = \frac{A_2-Y}{A_2(1-A_2)}$

3.) $\frac{\partial A_2}{\partial Z_2} = \frac{e^{-Z_2}}{(1+e^{-Z_2})^{2}} = \frac{(1-1+e^{-Z_2})}{(1+e^{-Z_2})^{2}} = \frac{1}{(1+e^{-Z_2})} \frac{(1-1+e^{-Z_2})}{(1+e^{-Z_2})} $ $ = \frac{1}{(1+e^{-Z_2})} [ \frac{(e^{-Z_2}+1)}{(1+e^{-Z_2})} - \frac{1}{(1+e^{-Z_2})}] = \frac{1}{(1+e^{-Z_2})} [ 1 - \frac{1}{(1+e^{-Z_2})}] = A_2(1-A_2)$

4.) $\implies \mathbf{\frac{\partial J}{\partial Z_2}} = \frac{A_2-Y}{A_2(1-A_2)} A_2(1-A_2) = \mathbf{A_2-Y};$ (subst. eqs 2 and 3 into 1)

5.) $ \mathbf{\frac{\partial J}{\partial Z_1}} = \frac{\partial J}{\partial Z_2} \frac{\partial Z_2}{\partial A_1} \frac{\partial A_1}{\partial Z_1} = ((A_2 - Y)^T W_2)^T \odot (1-tanh(Z_1)^2) = ((A_2 - Y)^T W_2)^T \odot (1-A_1^2)$

6.) $\frac{\partial J}{\partial W_2} = \frac{1}{n} (\frac{\partial J}{\partial Z_2} \frac{\partial Z_2}{\partial W_2}) = \frac{1}{n} ((A_2-Y) A_1^T)$

7.) $\frac{\partial J}{\partial B_2} = \frac{1}{n}\sum_{1}^{n}(\frac{\partial J}{\partial Z_2} \frac{\partial Z_2}{\partial B_2}) = \frac{1}{n}\sum_{1}^{n}(\frac{\partial J}{\partial Z_2} * 1) = \frac{1}{n} \sum_{1}^{n} (A_2-Y)$

8.) $\frac{\partial J}{\partial W_1} = \frac{1}{n}(\frac{\partial J}{\partial Z_1} \frac{\partial Z_1}{\partial W_1}) = \frac{1}{n}(\frac{\partial J}{\partial Z_1} X^T)$ $ = \frac{1}{n}((A_2 - Y)^T W_2)^T \odot (1-A_1^2) X^T$

9.) $\frac{\partial J}{\partial B_1} = \frac{1}{n}\sum_{1}^{n}(\frac{\partial J}{\partial Z_1} \frac{\partial Z_1}{\partial B_1}) = \frac{1}{n}\sum_{1}^{n}(\frac{\partial J}{\partial Z_1} * 1) = \frac{1}{n} \sum {\frac{\partial J}{\partial Z_1}} = \frac{1}{n} \sum(((A_2−Y)^TW_2)^T⊙(1−A_1^2)) $
